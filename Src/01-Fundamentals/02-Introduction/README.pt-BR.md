# Introdução ao C# (C-Sharp)

Este repositório contém uma visão geral descritiva sobre a linguagem de programação C#, cobrindo desde sua definição até os motivos para utilizá-la.

### O que é C#?

C# (pronuncia-se "C-Sharp") é uma linguagem de programação orientada a objetos criada pela Microsoft que roda sobre o .NET Framework. 

Alguns pontos importantes sobre sua origem e evolução:
*   **Raízes:** Possui raízes na família C e é muito próxima de outras linguagens populares como C++ e Java.
*   **Histórico:** A primeira versão foi lançada em 2002.
*   **Versão Atual:** A versão mais recente, C# 13, foi lançada em novembro de 2024.

### Para que o C# é usado?

A versatilidade do C# permite o desenvolvimento de uma vasta gama de soluções, incluindo:
*   Aplicações móveis e desktop.
*   Aplicações, sites e serviços web.
*   Desenvolvimento de Jogos e Realidade Virtual (VR).
*   Aplicações de banco de dados e muito mais.

### Por que usar C#?

*   **Popularidade:** É uma das linguagens de programação mais populares do mundo.
*   **Facilidade:** É fácil de aprender e simples de usar.
*   **Comunidade:** Possui um enorme suporte da comunidade global.
*   **Estrutura:** Por ser orientada a objetos, oferece uma estrutura clara aos programas e permite o reuso de código, reduzindo custos de desenvolvimento.
*   **Interoperabilidade:** Sua proximidade com C, C++ e Java facilita a transição de programadores entre essas linguagens.
